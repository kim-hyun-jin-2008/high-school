

## 🎒 왜 자료형이 따로 있을까?
문자로 **다 적을 수는 있지만**, 숫자랑 불린 같은 자료형이 따로 있는 이유는 **컴퓨터가 일을 더 빨리, 정확하게 하기 위해서**야. 초등학생도 이해할 수 있게 비유로 설명해볼게.

### 📌 1. **모두 글자로 써도 “뜻”을 구별하기 어렵기 때문**

예를 들어, `"10"`은 글자 1과 0을 나란히 쓴 **문자**야.
하지만 **숫자 10**은 계산할 수 있는 **진짜 숫자**야.

* `"10" + "10"` → `"1010"` (글자 이어 붙이기)
* `10 + 10` → `20` (계산)

둘이 완전히 다르지?
그래서 컴퓨터가 “이건 글자야?”, “이건 계산해야 해!” 하고 알아볼 필요가 있어.

---

### 📌 2. **컴퓨터가 빨리 계산하려면 숫자는 숫자대로 알아야 함**

컴퓨터는 계산을 엄청 빠르게 하는 기계야.
그런데 `"5"`처럼 글자로 쓰면 계산하기 전에
“이거 숫자 맞나?” 하고 또 확인해야 해.

🧮 진짜 숫자로 저장하면 → 바로 계산 가능
📄 글자로 저장하면 → 먼저 숫자로 바꾸고 계산해야 해서 느림

---

### 📌 3. **불린(Boolean)은 대답이 딱 두 개만 필요할 때 편함**

불린은 **true / false** 두 가지밖에 없어.
컴퓨터가 “맞아?” “틀려?” 같은 질문에 바로 쓸 수 있어.

예)

* `"true"`는 그냥 글자
* `true`는 진짜 “맞다”라는 뜻의 값

조건문에서 이렇게 쓸 수 있어:

```js
if (isAdult) {
  console.log("들어오세요!");
}
```

여기서 `isAdult`가 문자 `"true"`면 의미가 안 맞아.
하지만 `true`면 “진짜 참이네!” 하고 바로 판단할 수 있어.

---

## 🎯 한 줄로 정리
* **문자(string)**: 그냥 글자
* **숫자(number)**: 계산해야 할 값
* **불린(boolean)**: 맞다/틀리다 같은 판단용 값

컴퓨터가 헷갈리지 않고 빨리 일을 하게 하려고 자료형을 나눠 쓰는 거야!

# 📘 자료형(Data Types) 기초 설명

프로그래밍에서 **자료형**이란 “어떤 종류의 값인지”를 뜻합니다.
예를 들어, 숫자인지 문자(글자)인지, 참/거짓인지 등을 구분하는 것입니다.
자료형을 알아야 컴퓨터가 값을 제대로 이해하고 처리할 수 있습니다.

---

## 📋 기본 자료형 표

| 자료형             | 설명                           | 예시                             |
| --------------- | ---------------------------- | ------------------------------ |
| **숫자(Number)**  | 정수와 소수(실수)를 모두 포함하는 숫자 값         | `10`, `3.14`, `-5`             |
| **문자열(String)** | 글자나 문장을 표현하는 값. 자바스크립트에서는 문자가 없고 모두 문자열     | `"Hello"`, `'안녕하세요'`           |
| **불린(Boolean)** | 참(True) 또는 거짓(False)을 나타내는 값 | `true`, `false`                |

---

## 🧾 자료형별 쉽게 이해하기

### 1. **숫자(Number)**

컴퓨터가 계산할 수 있는 값들입니다.
정수, 소수 모두 같은 자료형입니다.

```js
let age = 20;
let pi = 3.14;

console.log("나이:", age);
console.log("원주율:", pi);

```

---

### 2. **문자열(String)**

글자, 문장 같은 텍스트를 표현합니다.
따옴표(`" "` 또는 `' '`) 안에 적습니다.

```js
let name = "홍길동";
let greeting = "안녕하세요!";

console.log("이름:", name);
console.log("인사말:", greeting);

```

---

### 3. **불린(Boolean)**

참(True) 또는 거짓(False)만 가지는 값입니다.
조건문에서 많이 쓰입니다.

```js
let isAdult = true;
let isStudent = false;

console.log("성인인가요?", isAdult);
console.log("학생인가요?", isStudent);

```
