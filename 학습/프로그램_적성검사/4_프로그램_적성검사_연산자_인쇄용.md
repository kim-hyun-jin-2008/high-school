# 연산자 종류 정리

연산자는 컴퓨터에게 **계산하라 / 비교하라 / 판단하라**고 말하는 도구예요.

---

## 1) 연산자 표 — 한눈에 보기

| 분류     |  연산자 | 무슨 뜻인지(쉬운 설명)                                                  | 코드 예시           | 출력 결과               |       |   |        |        |
| ------ | ---: | -------------------------------------------------------------- | --------------- | ------------------- | ----- | - | ------ | ------ |
| 사칙연산   |  `+` | 더하기 — 숫자들을 합쳐요.                                                | `3 + 2`         | `5`                 |       |   |        |        |
|        |  `-` | 빼기 — 앞에서 뒤를 빼요.                                                | `5 - 1`         | `4`                 |       |   |        |        |
|        |  `*` | 곱하기 — 숫자를 여러 번 더한 것과 같아요.                                      | `4 * 3`         | `12`                |       |   |        |        |
|        |  `/` | 나누기 — 나눠요.                                                     | `10 / 2`        | `5`                 |       |   |        |        |
| 나머지    |  `%` | 나머지 — 나누고 남은 숫자만 보여줘요.                                         | `11 % 2`        | `1`                 |       |   |        |        |
| 비교연산   | `==` | 같나? (**느슨 비교**) — 값만 같으면 `true`. 자료형(숫자/문자) 다르면 자동으로 바꿔서 비교해요. | `3 == "3"`      | `true`              |       |   |        |        |
|        | `!=` | 다르다? — 값이 다르면 `true`                                           | `3 != 4`        | `true`              |       |   |        |        |
|        |  `>` | 왼쪽이 더 크다?                                                      | `5 > 3`         | `true`              |       |   |        |        |
|        |  `<` | 왼쪽이 더 작다?                                                      | `2 < 7`         | `true`              |       |   |        |        |
|        | `>=` | 크거나 같다?                                                        | `5 >= 5`        | `true`              |       |   |        |        |
|        | `<=` | 작거나 같다?                                                        | `3 <= 4`        | `true`              |       |   |        |        |
| 논리연산   | `&&` | 그리고(AND) — 둘 다 참이어야 참                                          | `true && false` | `false`             |       |   |        |        |
|        |    ` |                                                                | `               | 또는(OR) — 하나라도 참이면 참 | `true |   | false` | `true` |
|        |  `!` | 아니다(NOT) — 참이면 거짓, 거짓이면 참                                      | `!true`         | `false`             |       |   |        |        |
| 할당(대입) |  `=` | 값을 넣는다 — 변수에 값을 저장해요.                                          | `let a = 5`     | (변수 a에 5 저장)        |       |   |        |        |

> 팁: `==`은 값만 비교해서 편리하지만 때로는 뜻밖의 결과가 나와요. (예: `0 == false` → `true`) 그래서 큰 프로그램에서는 조심해야 해요. 하지만 지금은 `==`을 익혀보고 값이 같은지 확인하는 연습으로 좋아요.

## 2) 짧고 친절한 설명

* **사칙연산 (+ - * /)**: 수학 시간에 하던 계산 버튼이에요.
* **% (나머지)**: 나눴을 때 남는 것만 꺼내요. (예: 과자 11개를 2명씩 나누면 1개 남음)
* **비교연산 (==, >, < 등)**: `맞아? 틀려?` 물어보는 질문이에요. 답은 `true`(참) 또는 `false`(거짓).

  * `==`은 값만 보고 비교해요. 즉 숫자 3과 문자열 "3"도 같다고 판단할 수 있어요.
* **논리연산 (&&, ||, !)**: 여러 조건을 합칠 때 써요.

  * `&&`는 둘 다 해야 참, `||`는 둘 중 하나만 해도 참, `!`는 반대로 뒤집어요.
* **=`(대입)`**: 변수라는 상자에 값을 넣는 행동이에요.

## 3) Node.js `console.log` 예제 (그대로 복사해서 실행 가능)

```javascript
// === 연산자 예제 모음 ===
console.log("=== 사칙연산 ===");
console.log("3 + 2 =", 3 + 2);
console.log("10 - 4 =", 10 - 4);
console.log("5 * 2 =", 5 * 2);
console.log("20 / 4 =", 20 / 4);
console.log("11 % 2 =", 11 % 2); // 나머지

console.log("\n=== 비교연산 (== 사용 예제) ===");
console.log('3 == "3" ->', 3 == "3"); // 값 같음 -> true
console.log("5 == 5 ->", 5 == 5);
console.log("3 != 4 ->", 3 != 4);
console.log("5 > 3 ->", 5 > 3);
console.log("2 < 7 ->", 2 < 7);
console.log("5 >= 5 ->", 5 >= 5);
console.log("3 <= 4 ->", 3 <= 4);

console.log("\n=== 논리연산 ===");
console.log("true && false ->", true && false);
console.log("true || false ->", true || false);
console.log("!true ->", !true);

// 간단한 상황 예제: 게임 규칙
console.log("\n=== 상황 예제 ===");
let score = 95;
let hasHomework = false;
console.log("점수:", score);
console.log("합격(점수 >= 100)?", score >= 100); // 비교
console.log("게임 가능(숙제 했거나 점수 90 이상)?", hasHomework || score >= 90); // 논리
```

### 예상 출력 (보이는 대로)

```
=== 사칙연산 ===
3 + 2 = 5
10 - 4 = 6
5 * 2 = 10
20 / 4 = 5
11 % 2 = 1

=== 비교연산 (== 사용 예제) ===
3 == "3" -> true
5 == 5 -> true
3 != 4 -> true
5 > 3 -> true
2 < 7 -> true
5 >= 5 -> true
3 <= 4 -> true

=== 논리연산 ===
true && false -> false
true || false -> true
!true -> false

=== 상황 예제 ===
점수: 95
합격(점수 >= 100)? false
게임 가능(숙제 했거나 점수 90 이상)? true
```

## 4) 한 줄 요약

* 연산자는 컴퓨터한테 **계산해!**, **비교해!**, **조건 확인해!** 라고 말하는 도구예요.
* `==`은 **값만 같으면** 같은 것으로 판단해요(숫자와 문자도 같다고 판단할 수 있어요).
* 직접 `console.log`로 실행해보면 어떤 결과가 나오는지 바로 확인할 수 있어요.

원하면 위 예제를 더 간단한 문제들(퀴즈형)로 바꿔서 연습문제로 만들어줄게!
