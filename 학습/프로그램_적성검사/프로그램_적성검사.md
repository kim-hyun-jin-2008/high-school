### 변수와 상수
변수와 상수의 개념을 이해야 햐는 곳입니다.
* 변수: 어떤 값을 가지고 있는데 다른 값을 넣을 수 있는것. 즉, 변하는것
* 상수: 어떤 값을 가지고 있는데 다른 값을 넣을 수 **없는것**. 즉, 변하지 않는것

### 배열
일반적인 변수, 상수는 1개의 값만 넣을 수 있지만 여러개의 값을 넣어야 할때 사용합니다. 1차원, 2차원, 3차원까지 보통 많이 사용 하고 어떤 개념인지 알아야 합니다.


### 함수
