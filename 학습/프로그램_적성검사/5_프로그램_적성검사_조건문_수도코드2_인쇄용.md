# 🧠 1. 수도코드(글로 정리한 단계)

```
1. 물건 가격을 저장한다.
2. 할인율을 저장할 변수를 0으로 준비한다.

3. 만약 가격이 100000원 이상이라면
      → 할인율을 20%로 정하고
      → "20% 할인이 적용됩니다." 라고 출력한다.

4. 그렇지 않고 가격이 50000원 이상이라면
      → 할인율을 10%로 정하고
      → "10% 할인이 적용됩니다." 라고 출력한다.

5. 위 조건 둘 다 아니면
      → 할인율을 0%로 정하고
      → "할인이 없습니다." 라고 출력한다.

6. 최종 가격을 계산한다.
      → 원래 가격에서 (가격 × 할인율 ÷ 100) 만큼 뺀다.

7. 마지막으로 최종 가격을 출력한다.
```

# 💻 2. 실제 JavaScript 코드

```javascript
console.log("\n=== 예제 6: 할인율 계산 ===");

let price = 50000;
let discount = 0;

if (price >= 100000) {
    discount = 20;
    console.log("20% 할인이 적용됩니다.");
} else if (price >= 50000) {
    discount = 10;
    console.log("10% 할인이 적용됩니다.");
} else {
    discount = 0;
    console.log("할인이 없습니다.");
}

let finalPrice = price - (price * discount / 100);
console.log("최종 가격: " + finalPrice + "원");
```

# 🔍 3. 논리적인 사고 설명

### ✔ 논리적 사고란?

문제를 해결할 때
**조건을 하나씩 정확히 따져보고, 어떤 순서로 검사해야 하는지 판단하는 과정**이야.

---

## ✔ 이 문제에서 논리적 사고가 필요한 이유

### ① 조건의 "순서"가 중요하기 때문

예를 들어, 아래처럼 잘못 배열하면 문제가 생겨.

❌ 먼저 `price >= 50000`을 검사하면?
100000원인 경우도 이 조건에서 멈춰버려서 **20%가 아닌 10% 할인**이 되어버림.

그래서 조건은 아래처럼 **큰 금액부터 검사해야 정확함**

* price ≥ 100000 → 20%
* price ≥ 50000 → 10%
* 나머지 → 0%

---

### ② 모든 경우를 빠짐없이 고려해야 함

가격이

* 10만원 이상일 때
* 5만원 이상일 때
* 그 외일 때

이렇게 세 가지 경우를 모두 고려하고
겹치지 않게 나누는 것이 **논리적 분류**야.

---

### ③ 계산하기 전에 “정확한 값”을 먼저 만들어야 함

할인율을 정확히 결정한 다음
최종 가격을 계산해야 실수가 없어.

즉,

1. 판단
2. 결정
3. 계산
4. 출력
   이 순서가 완전히 맞아야 원하는 결과가 나와.
