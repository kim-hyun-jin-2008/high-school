
# 📘 자료형(Data Types) 기초 설명

프로그래밍에서 **자료형**이란 “어떤 종류의 값인지”를 뜻합니다.
예를 들어, 숫자인지 문자(글자)인지, 참/거짓인지 등을 구분하는 것입니다.
자료형을 알아야 컴퓨터가 값을 제대로 이해하고 처리할 수 있습니다.

---

## 📋 기본 자료형 표

| 자료형             | 설명                           | 예시                             |
| --------------- | ---------------------------- | ------------------------------ |
| **숫자(Number)**  | 정수와 소수(실수)를 모두 포함하는 숫자 값         | `10`, `3.14`, `-5`             |
| **문자열(String)** | 글자나 문장을 표현하는 값. 자바스크립트에서는 문자가 없고 모두 문자열     | `"Hello"`, `'안녕하세요'`           |
| **불린(Boolean)** | 참(True) 또는 거짓(False)을 나타내는 값 | `true`, `false`                |
| **undefined**   | 값이 아직 정의되지 않은 상태. 이건 오류(Error)             | 값이 할당되지 않은 변수                  |
| **null**        | 값이 “없다”는 것을 의도적으로 표현         | `null`                         |
| **객체(Object)**  | 여러 값을 하나로 묶은 구조              | `{ name: "Tom", age: 20 }`     |
| **배열(Array)**   | 여러 데이터를 순서대로 저장한 목록          | `[1, 2, 3]`, `["a", "b", "c"]` |

---

## 🧾 자료형별 쉽게 이해하기

### 1. **숫자(Number)**

컴퓨터가 계산할 수 있는 값들입니다.
정수, 소수 모두 같은 자료형입니다.

```js
let age = 20;
let pi = 3.14;

console.log("나이:", age);
console.log("원주율:", pi);

```

---

### 2. **문자열(String)**

글자, 문장 같은 텍스트를 표현합니다.
따옴표(`" "` 또는 `' '`) 안에 적습니다.

```js
let name = "홍길동";
let greeting = "안녕하세요!";

console.log("이름:", name);
console.log("인사말:", greeting);

```

---

### 3. **불린(Boolean)**

참(True) 또는 거짓(False)만 가지는 값입니다.
조건문에서 많이 쓰입니다.

```js
let isAdult = true;
let isStudent = false;

console.log("성인인가요?", isAdult);
console.log("학생인가요?", isStudent);

```

---

### 4. **undefined**

값을 **아직 넣지 않은 상태**입니다.

```js
let x;
console.log("x의 값:", x); // undefined 출력

```

---

### 5. **null**

값이 **없다**는 것을 개발자가 명확하게 표현할 때 사용합니다.

```js
let data = null;
console.log("data의 값:", data); // null 출력

```

---

### 6. **객체(Object)**
여러 데이터를 하나의 묶음으로 관리할 수 있습니다. 사람 정보처럼 여러 속성이 있는 데이터에 적합합니다.

```js
let user = {
  name: "Tom",
  age: 25,

};

// 객체를 사용하지 않으면 아래 처럼 사용해야 하는 불편함이 있고. 수정(유지보수)를 생각해 보면 좋다.
// let 홍길동_얼굴 = '동그람';
// let 홍길동_성별 = '여성';
// let 홍길동_성격 = '좋음';

let 이순신 = {
  얼굴: '동그람',
  귀  : '눌림',
  눈  : '이쁨',
  성격: '좋음',
  성향: '낙천적'
  성별: '여성'
};

// 아래 오류 발생 하는것은 위 객체 변수명을 보고 올바르게 변경해 출력해 보세요.
console.log("사용자 객체:", user);
console.log("이름:", user.name);
console.log("나이:", user.age);
console.log("명수진 성별:", 이순신.성별);

```

---

### 7. **배열(Array)**

데이터들을 순서대로 담아놓은 상자 같은 구조입니다.
**0번부터** 시작합니다.

- 인덱스 번호인데. 이건 쉽게 방번호라고 생각하면 되고.
* 몇 번째 있는것인지 꺼내 올려면 지정을 해야 하니까 번호가 필요하지.
+ 번호가 아니라도 어떤 것이라도 어디에 있는지 어떤 구분은 있어야 하는데 배열에선 그걸 숫자를 사용.

```js
let colors = ["red", "blue", "green"];
let numbers = [1, 2, 3, 4, 5]; // 값만 들어간것. 배열 인덱스는 이런식으로 하면 자동으로 0부터 시작

// let colors[4] = 5; // 이렇게 하면 오류가 발생 한다.

// 추가로 넣고 싶으면 순서대로 넣어 주면 된다.
// 4번째(인덱스 3)에 값 추가 (배열의 길이보다 큰 인덱스에 값 할당 가능)
colors[3] = "yellow"; // 인덱스 3에 "yellow" 추가
colors[4] = "purple"; // 인덱스 4에 "purple" 추가



console.table(colors);
console.log("색깔 배열:", colors);
console.log("첫 번째 색:", colors[0]);

console.table(numbers);
console.log("숫자 배열:", numbers);
console.log("세 번째 숫자:", numbers[2]);

```

정수/실수, 문자/문자열까지 포함한 전체 정리 + 실습 예제입니다.

# 📌 1. 정수 vs 실수

JavaScript에서는 정수·실수 모두 **Number** 하나로 통합되어 있습니다.

## ✔ 비교 표

| 구분    | 정수(Integer)     | 실수(Float)      |
| ----- | --------------- | -------------- |
| 예시    | `10`, `0`, `-3` | `3.14`, `0.25` |
| 자료형   | number          | number         |
| 구분 여부 | ❌ 없음            | ❌ 없음           |
| 특징    | 소수점 없음          | 소수점 있음         |

## ✔ 실습 예제

```js
let a = 10;   // 정수
let b = 3.14; // 실수

console.log("a =", a, ", type =", typeof a);
console.log("b =", b, ", type =", typeof b);
```

---

# 📌 2. 문자 vs 문자열

JavaScript에는 **문자(char) 타입이 존재하지 않음**
→ 모든 글자는 **문자열(String)**로 저장됨.

## ✔ 비교 표

| 구분  | 문자(Char) | 문자열(String) |
| --- | -------- | ----------- |
| 예시  | `'A'`    | `"Hello"`   |
| 자료형 | ❌ 없음     | ✔ string    |
| 차이  | 없음       | 문자열만 존재     |

## ✔ 실습 예제

```js
let ch = "A";            // 문자처럼 보이지만 문자열
let text = "Hello World";

console.log(ch, "type =", typeof ch);
console.log(text, "type =", typeof text);
```

---

# 📌 3. 작은따옴표 · 큰따옴표 · 백틱 비교

JavaScript 문자열은 세 가지 방식으로 만들 수 있습니다:
**'작은따옴표', "큰따옴표", `백틱`**

## ✔ 비교 표

| 구분        | 작은따옴표 `' '`    | 큰따옴표 `" "`      | 백틱 `` ` ` ``        |
| --------- | -------------- | --------------- | ------------------- |
| 용도        | 문자열            | 문자열             | 문자열                 |
| 기능 차이     | 없음             | 없음              | ✔ 템플릿 리터럴(변수 삽입 가능) |
| 내부 따옴표 처리 | 내부에 큰따옴표 넣기 편함 | 내부에 작은따옴표 넣기 편함 | 모두 자유로움             |
| 줄바꿈       | ❌ 안 됨          | ❌ 안 됨           | ✔ 가능                |
| 변수 포함     | ❌ 불가           | ❌ 불가            | ✔ `${변수}` 사용        |
| 자료형       | string         | string          | string              |

---

## ✔ 실습 예제

```js
let s1 = 'Hello';
let s2 = "안녕하세요";
let name = "Tom";

// 백틱의 장점: 변수 삽입
let s3 = `Hello, ${name}`;
let s3 = "Hello, " + name; // 위하고 같은것

let 카테고리명 = '복합기업';
let 설명용 = `
 <tr>
        <td>${카테고리명}</td>
        <td>2025-11-11 &nbsp; </td>
        <td><a href='#' onclick="loadStockData('000150', '두산'); return false;">000150 두산 </a>(1,256,667)</td>
        <td>35.42%</td>
        <td align='center'>9</td>
        </tr>
`;

let 설명용 = "\n";
설명용 += "<tr>\n";
설명용 += "<td>" + 카테고리명 + "</td>\n";
설명용 += "<td>2025-11-11 &nbsp; </td>\n";
설명용 += "<td><a href='#' onclick="loadStockData('000150', '두산'); return false;">000150 두산 </a>(1,256,667)</td>\n";
설명용 += "<td>35.42%</td>\n";
설명용 += "<td align='center'>9</td>\n";
설명용 += "</tr>\n";



// 백틱의 장점: 여러 줄 문자열
let s4 = `
여러 줄을
그대로 쓸 수 있습니다.
`;

console.log(s1);
console.log(s2);
console.log(s3);
console.log(s4);

// 작은따옴표/큰따옴표 내부 처리
let quote1 = '그는 "안녕"이라고 말했다.';
let quote2 = "I'm a developer.";

console.log(quote1);
console.log(quote2);
```

백틱(`` ```)이 가진 기능들을 초보자도 한눈에 이해할 수 있도록 구성했습니다.

# 📌 템플릿 문자열(Template Literal) 상세 설명 표

## ✔ 템플릿 문자열 개요

템플릿 문자열은 **백틱(`` ```)** 으로 감싸서 만드는 문자열이며
일반 문자열로는 불편했던 기능들을 편하게 해결할 수 있는 문법입니다.

---

## ✔ 템플릿 문자열 기능 상세 비교 표

| 항목                       | 설명                                      | 예시                                         |
| ------------------------ | --------------------------------------- | ------------------------------------------ |
| **기본 표기법**               | 백틱(`` ```)으로 문자열을 감쌈                    | `` const msg = `Hello`; ``                |
| **변수 삽입(Interpolation)** | `${변수}` 형태로 문자열 사이에 변수·값을 자연스럽게 넣을 수 있음 | `` `Hi, ${name}` `` |
| **표현식 삽입**               | 단순 변수뿐 아니라 간단한 계산, 함수 호출도 가능            | `` `10 + 5 = ${10 + 5}` ``              |
| **여러 줄 문자열(Multiline)**  | 줄바꿈을 그대로 표현할 수 있어 `\n` 없이도 줄이 나뉨        | `` const t = `line1\nline2`; `` (자동 줄바꿈)  |
| **내부 따옴표 자유롭게 사용 가능**    | 작은·큰따옴표를 escape 없이 자연스럽게 포함 가능          | `` `He said "Hi" and I'm fine.` ``       |
| **HTML 템플릿 작성 용이**       | 웹 개발 시 긴 HTML을 가독성 좋게 작성 가능             | `` const html = `<div>${title}</div>`;`` |
| **문자열 + 값 연결 가독성↑**      | 일반 문자열의 `+` 연산보다 훨씬 깔끔하고 실수 감소          | `` `User: ${name}, Age: ${age}` ``      |
| **타입**                   | 결과값은 일반 문자열과 동일하게 `string`              | `` typeof `Hello` → "string" ``       |

---

# 📘 템플릿 문자열 심화 실습 예제

## 1) 변수 삽입 기본

```js
let name = "Tom";
console.log(`Hello, ${name}!`);
```

---

## 2) 표현식 삽입

```js
console.log(`3 + 5 = ${3 + 5}`);
```

---

## 3) 여러 줄 문자열

```js
let text = `
이 문장은
줄바꿈이 그대로
유지됩니다.
`;

console.log(text);
```

---

## 4) HTML 템플릿 작성

```js
let title = "Welcome";
let html = `
<div class="box">
  <h1>${title}</h1>
  <p>This is a template literal example.</p>
</div>
`;

console.log(html);
```

---

## 5) 따옴표 문제 없이 사용
아래는 큰따음표, 작은따음표 상관 없이 백틱안에서는 마음대로 써도 된다.

```js
let quote = `그는 "여기 좋다"라고 말했고, 나는 '응 맞아'라고 답했다.`;
console.log(quote);
```
