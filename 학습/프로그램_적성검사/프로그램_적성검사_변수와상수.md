# 프로그램 적성 확인용 - 변수와상수
강의 내용을 얼마나 잘 이해했고 논리적으로 사고 하는지 알아 보기 위한 것입니다.

### 변수와 상수
변수와 상수의 개념을 이해야 햐는 곳입니다. 자바스크립트는 자동변수 이지만 숫자, 문자형의 구분은 실제 프로그램에선 상당히 중요한 부분입니다.
* 변수: 어떤 값을 가지고 있는데 다른 값을 넣을 수 있는것(덮어 씌운다은 의미 - 기존것은 없어진다). 즉, 변하는것
  + 숫자형 변수: 4칙연산등 테스트 해보도록 합니다.
  + 문자형 변수: 문자열을 출력해보고. 문자열을 붙여 봅니다.
    - 일반적인 언어에서는: `문자, "문자열 이렇게 사용하지만 javascript는 구분하지 않는다.
  + 배열 - 뒤쪽에서 배우게 됩니다.

* 상수: 어떤 값을 가지고 있는데 다른 값을 넣을 수 **없는것**. 즉, 변하지 않는것


## nodeJS 변수와 상수
유동성(융통성) 있게 프로그램을 작성하기 위해서 가장 기초적으로 알아야 할것이 변수와 상수 입니다.

* 변수 (Variable)
  - let: 값을 변경할 수 있는 변수 선언.
    + 난 이걸 이렇게 하겠다고 하는것이 선언 하는것입니다. let a = 5; 라고 하면 난 a에 5라는 값을 넎는 것을 선언 한것입니다.

* 상수 (Constant)
  - const: 값을 변경할 수 없는 상수 선언
  - 변경하면 안되는 값을 저장해야 된다면 이것을 사용 합니다.
    + 예를 들면 선풍기 1단, 2단, 3단 이 있을 경우 1단을 누르면 1초에 10바귀, 2단을 누르면 1초에 20바퀴
    + 이런 형태로 각 단수에 따라서 변경 되면 안되는 경우 상수로 지정 합니다. 1단 눌렀는데 어떤때는 20바퀴 어떤때는 30바퀴 이러면 사용하는데 있어 예측성이 없기 때문에 그렇습니다.


#### 연습문제

* 1. 변수 선언과 재할당
```javascript
// let으로 변수 선언
let userName = "홍길동";
console.log("초기 이름:", userName, userName);

// 변수 값 변경 가능
userName = "김철수";
console.log("변경된 이름:", userName);

let age = 25;
console.log("나이:", age);
age = 26;
console.log("변경된 나이:", age);
```

* 2. 상수 선언
```javascript
// const로 상수 선언
const PI = 3.14159;
console.log("원주율:", PI);

const COMPANY_NAME = "테크컴퍼니";
console.log("회사명:", COMPANY_NAME);
```

* 3. 데이터 타입
```javascript
// 문자열 (String)
let greeting = "안녕하세요";
console.log("문자열:", greeting, "- 타입:", typeof greeting);

// 숫자 (Number)
let score = 100;
console.log("숫자:", score, "- 타입:", typeof score);
```

* 4. 숫자형
```javascript
// 정수
let age = 25;
console.log("나이:", age);

// 실수
let height = 175.5;
console.log("키:", height);

// 음수
let temperature = -10;
console.log("온도:", temperature);

// 숫자 연산
let num1 = 10;
let num2 = 5;
let result;

result = num1 + num2;
console.log("덧셈:", num1, "+", num2, "=", result);

result = num1 - num2;
console.log("뺄셈:", num1, "-", num2, "=", result);

result = num1 * num2;
console.log("곱셈:", num1, "*", num2, "=", result);

result = num1 / num2;
console.log("나눗셈:", num1, "/", num2, "=", result);
```

* 5. 변수명 규칙
```javascript
// 올바른 변수명 (camelCase 방식만 사용)
let userName = "홍길동";

console.log("올바른 변수명 예제:");
console.log("  userName:", userName);
console.log("  userAge:", userAge);
console.log("  userCity:", userCity);

// 잘못된 변수명 (주석 처리)
// let 2age = 25; // 숫자로 시작 불가
// let user-name = "홍길동"; // 하이픈 사용 불가
// let let = "value"; // 예약어 사용 불가
```

* 6. 블록 스코프
여기선 몰라도 되기 때문에 배우지 않지만 예를 들면 다음과 같습니다.
  - 이 문제는 프로그램 하다 보면 변수명을 같은 것을 많이 사용하게 되는데 그럴때 이전 선언한것에 아래쪽에서 덮어 씌운줄 모르고 처리 하는 경우등 때문에 잘 사용하면 좋은 부분 입니다.
  - 함수를 사용해서 별도 관리 하는 방법도 있습니다.
  - 보통은 중간 중간 같은 이름이 있을것 같으면 Ctrl+F로 찾아서 확인해 보거나 문제 발생 시키고 싶지 않으면 초기화 시켜 사용 하며, 각각 위치에 따라 접두어 같은것을 주는 방법도 있습니다.

```javascript
let outerVar = "외부 변수";
const outerConst = "외부 상수";

if (true) {
  let innerVar = "내부 변수";
  const innerConst = "내부 상수";

  console.log("블록 내부:");
  console.log("  -", outerVar);
  console.log("  -", outerConst);
  console.log("  -", innerVar);
  console.log("  -", innerConst);
}

console.log("\n블록 외부:");
console.log("  -", outerVar);
console.log("  -", outerConst);
// console.log("  -", innerVar); // ReferenceError 발생
```
