# 프로그램 적성확인 - 조건문
특정 조건의 참과 거짓을 평가하여 상황에 따라 서로 다른 코드를 실행함으로써 프로그램이 다양한 입력과 상황에 유연하게 대응하고 적절한 의사 결정을 내릴 수 있도록 하는 것입니다.

## 연산자
### 4칙연산자
  - 기본 산술 연산 (+, -, *, /)
  - 나머지 연산 (%)

| 연산자 | 이름 | 설명 | 예제 | 결과 |
|--------|------|------|------|------|
| + | 덧셈 | 두 값을 더합니다 | 10 + 5 | 15 |
| - | 뺄셈 | 첫 번째 값에서 두 번째 값을 뺍니다 | 10 - 5 | 5 |
| * | 곱셈 | 두 값을 곱합니다 | 10 * 5 | 50 |
| / | 나눗셈 | 첫 번째 값을 두 번째 값으로 나눕니다 | 10 / 5 | 2 |
| % | 나머지 | 나눗셈의 나머지를 구합니다 | 10 % 3 | 1 |

### 비교연산자
  - 일치/불일치 (===, !==)
  - 크기 비교 (>, <, >=, <=)

| 연산자 | 이름 | 설명 | 예제 | 결과 |
|--------|------|------|------|------|
| === | 일치 | 값과 타입이 모두 같은지 비교 | 10 === 10 | true |
| !== | 불일치 | 값 또는 타입이 다른지 비교 | 10 !== 5 | true |
| > | 크다 | 왼쪽 값이 오른쪽 값보다 큰지 비교 | 10 > 5 | true |
| < | 작다 | 왼쪽 값이 오른쪽 값보다 작은지 비교 | 10 < 5 | false |
| >= | 크거나 같다 | 왼쪽 값이 오른쪽 값보다 크거나 같은지 비교 | 10 >= 10 | true |
| <= | 작거나 같다 | 왼쪽 값이 오른쪽 값보다 작거나 같은지 비교 | 10 <= 5 | false |

### 논리연산자
  - AND (&&)  : 교집합
    ```javascript
    true && true   // true
    true && false  // false
    false && true  // false
    false && false // false
    ```

  - OR (||)   : 합집합
    ```javascript
    true || true   // true
    true || false  // true
    false || true  // true
    false || false // false
    ```
  - NOT (!)   : 부정
    ```javascript
    !true  // false
    !false // true
    ```

| 연산자 | 이름 | 설명 | 예제 | 결과 |
|--------|------|------|------|------|
| && | AND (그리고) | 두 조건이 모두 참이면 참 | true && true | true |
| \|\| | OR (또는) | 두 조건 중 하나라도 참이면 참 | true \|\| false | true |
| ! | NOT (부정) | 참을 거짓으로, 거짓을 참으로 변경 | !true | false |





## 조건문
* if: 조건이 참일 때만 실행
* if-else: 참/거짓 두 가지 경우
* if-else if-else: 여러 조건을 순서대로 체크

### 실습내용
* if 문: 단일 조건 체크
```javascript
  console.log("=== 1. if 문 (단일 조건) ===\n");

  let age = 20;

  if (age >= 18) {
    console.log("성인입니다.");
  }

  let score = 85;

  if (score >= 80) {
    console.log("점수가 80점 이상입니다.");
  }

  let isRaining = true;

  if (isRaining) {
    console.log("우산을 가져가세요.");
  }
```

* if-else 문: 두 가지 경우 나누기
```javascript
  let temperature = 25;

  if (temperature >= 30) {
    console.log("날씨가 덥습니다.");
  } else {
    console.log("날씨가 시원합니다.");
  }

  let number = 7;

  if (number % 2 === 0) {
    console.log(number + "는 짝수입니다.");
  } else {
    console.log(number + "는 홀수입니다.");
  }

  let password = "1234";

  if (password === "1234") {
    console.log("로그인 성공!");
  } else {
    console.log("비밀번호가 틀렸습니다.");
  }
```

* if-else if-else 문: 여러 조건 체크 (학점, 시간대, 나이별 분류)

  let time = 14;

    console.log("오후입니다.");

* 비교 연산저
```javascript
  let a = 10;
  let b = 20;

  console.log("a = " + a + ", b = " + b);
  console.log("");

  if (a > b) {
    console.log("a가 b보다 큽니다.");
  }

  if (a < b) {
    console.log("a가 b보다 작습니다.");
  }

  if (a === b) {
    console.log("a와 b가 같습니다.");
  }

  if (a !== b) {
    console.log("a와 b가 다릅니다.");
  }

  if (a <= 10) {
    console.log("a는 10 이하입니다.");
  }

  if (b >= 20) {
    console.log("b는 20 이상입니다.");
  }
```

* 논리연산자
```javascript
  let korean = 80;
  let math = 90;

  // AND 연산자 (&&) - 둘 다 참이어야 참
  if (korean >= 80 && math >= 80) {
    console.log("국어와 수학 모두 80점 이상입니다.");
  }

  // OR 연산자 (||) - 하나만 참이면 참
  if (korean >= 90 || math >= 90) {
    console.log("국어 또는 수학이 90점 이상입니다.");
  }

  // NOT 연산자 (!) - 참을 거짓으로, 거짓을 참으로
  let isSunny = false;

  if (!isSunny) {
    console.log("날씨가 맑지 않습니다.");
  }
```

### 실습 예제 - 3
* 예제 11: 영화 관람 등급 판별
  - example11.js로 저장
  - node example11.js 실행
  - 나이를 입력하면 볼 수 있는 영화 등급 판별

```javascript
// 예제 11: 나이 입력받아 영화 관람 등급 판별
// 실행 방법: node example11.js

const readline = require("readline");

const rl = readline.createInterface({
    input: process.stdin,
    output: process.stdout
});

console.log("=== 영화 관람 등급 판별 ===");
rl.question("나이를 입력하세요: ", input1 => {
    let inputAge = parseInt(input1);

    if (inputAge >= 19) {
        console.log("청소년 관람불가 영화를 볼 수 있습니다.");
    } else if (inputAge >= 15) {
        console.log("15세 이상 관람가 영화를 볼 수 있습니다.");
    } else if (inputAge >= 12) {
        console.log("12세 이상 관람가 영화를 볼 수 있습니다.");
    } else {
        console.log("전체 관람가 영화만 볼 수 있습니다.");
    }

    rl.close();
});
```

* 예제 12: 시험 합격/불합격 판정
  - example12.js로 저장
  - node example12.js 실행
  - 점수를 입력하면 합격 여부와 점수 차이 계산

```javascript
// 예제 12: 점수 입력받아 합격/불합격 판정
// 실행 방법: node example12.js

const readline = require("readline");

const rl = readline.createInterface({
    input: process.stdin,
    output: process.stdout
});

console.log("=== 시험 합격/불합격 판정 ===");
rl.question("시험 점수를 입력하세요 (0-100): ", input2 => {
    let examScore = parseInt(input2);
    const passingScore = 60;

    if (examScore >= 0 && examScore <= 100) {
        if (examScore >= passingScore) {
            let extraPoints = examScore - passingScore;
            console.log("합격입니다! 축하합니다!");
            console.log("합격선보다 " + extraPoints + "점 높습니다.");
        } else {
            let lackPoints = passingScore - examScore;
            console.log("불합격입니다. 다음 기회에 도전하세요!");
            console.log("합격선까지 " + lackPoints + "점 부족합니다.");
        }
    } else {
        console.log("잘못된 점수입니다. 0-100 사이의 점수를 입력하세요.");
    }

    rl.close();
});
```
